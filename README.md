## FAQ

**1) Does this setup enable AES-256 by default?**  
Yes—AES-256 encryption is part of the baseline configuration. Focus on enabling Kill Switch and picking the right server.

**2) What’s the first setting I should turn on?**  
**Kill Switch.** It’s the difference between “VPN dropped” and “VPN dropped and leaked.”

**3) How do I get the best speed?**  
Pick the closest server in **Global Servers**, avoid overloaded regions, and keep your protocol on the default unless you know why you’re changing it.

**4) Is a No-Logs policy enough for privacy?**  
It’s a core piece, but you should also keep Kill Switch enabled and avoid mixing VPN traffic with sketchy browser extensions.
